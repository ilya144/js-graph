# Js-graph

https://js-graph-demo.firebaseapp.com/

## Запуск тестового сервера
```
npm install
npm run start
```

## Cборкa проекта
```
npm install
npm run build
```

## Преобразование excel в json 
совместимый с приложением
```
python3 -m pip install xlrd
python3 xlsx_json.py graph.xlsx
mv graph.xlsx.json public/
```
